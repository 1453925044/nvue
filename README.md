# uni-app(nvue)
基于uni-app的nuve登录模板，支持android、ios、微信小程序、h5，样式丰富，app支持自定义状态栏、导航栏，以及调用第三方授权登录(QQ、微信、微博、支付宝)\
支付宝三方登录授权请自己集成第三方SDK\
开箱即用，适合市场上大多数登录方式\
